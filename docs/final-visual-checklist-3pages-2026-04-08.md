## 1) `/sales/material` 영업자료실
### 체크 항목
- [ ] 페이지 타이틀/설명 위치가 다른 페이지와 톤이 동일하다.
- [ ] 검색창과 `등록` 버튼이 한 줄 정렬되고 높이가 맞다.
- [ ] 테이블 헤더(`제목/유형/등록일/첨부`) 정렬이 맞다.
- [ ] `다운로드` 버튼 텍스트가 세로가 아닌 가로로 보인다.
- [ ] 첨부 없음(`없음`) 표기 위치가 행 중앙 정렬이다.
- [ ] 행 hover 시 배경 변화가 자연스럽고 깨짐이 없다.

### 스크린샷
- [ ] 전체 화면
- [ ] 첨부 컬럼 확대

## 2) `/master/performance-plan` 성과 계획 관리
### 체크 항목
- [ ] 우상단 액션 버튼(`전년 복사/엑셀 양식 다운로드/저장`) 기준선이 맞다.
- [ ] 조회 필터 영역과 상단 액션 영역 사이 간격이 일정하다.
- [ ] 테이블 텍스트가 세로로 꺾이지 않고 가로로 보인다.
- [ ] 월 입력칸 숫자 4~5자리가 잘리지 않는다.
- [ ] 가로 스크롤 시 헤더/본문 경계가 어긋나지 않는다.
- [ ] 상세 모달 오픈 시 본문/이력 목록 레이아웃이 무너지지 않는다.

### 스크린샷
- [ ] 전체 화면
- [ ] 테이블 + 우상단 버튼 영역 확대

## 3) `/partner/notice` 공지/자료실
### 체크 항목
- [ ] 탭/필터/테이블 카드 스타일이 프로젝트 공통 톤과 맞다.
- [ ] 긴 제목 표시 시 줄바꿈/툴팁/행 높이가 자연스럽다.
- [ ] 상세 모달에서 긴 본문 스크롤이 자연스럽다.
- [ ] 첨부파일 목록 링크 간격/정렬이 깨지지 않는다.
- [ ] 등록 모달 폼(유형/기간/내용/첨부) 간격이 균일하다.
- [ ] 모달 닫기/배경 클릭 시 의도대로 닫힌다.

### 스크린샷
- [ ] 목록 화면
